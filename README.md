👥 Amigo Secreto (Meu Primeiro Projeto de Lógica)
Este foi o primeiro projeto que desenvolvi durante a minha formação em Lógica de Programação na Alura. Ele representa o marco inicial da minha jornada como desenvolvedor, onde apliquei pela primeira vez conceitos fundamentais de programação estruturada.
🎯 O Desafio Inicial
O objetivo era criar uma aplicação que gerenciasse uma lista de amigos e realizasse um sorteio aleatório, garantindo que ninguém tirasse a si mesmo e que o fluxo de dados fosse consistente.
💡 O que aprendi com este projeto:
• Manipulação de Arrays: Como adicionar, armazenar e percorrer listas de nomes.
• Lógica de Sorteio: Implementação do algoritmo para embaralhar e selecionar resultados de forma aleatória.
• Validações de Input: Aprendi a importância de tratar erros (como impedir campos vazios ou nomes duplicados).
• Interação com o DOM: Como fazer o JavaScript "conversar" com o HTML para exibir os resultados na tela em tempo real.
🛠️ Tecnologias
• JavaScript: Toda a inteligência e lógica do sorteio.
• HTML5/CSS3: Estrutura e estilização da interface.
